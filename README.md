# Sudoku Solver (Python)

A Python Sudoku solver implemented with backtracking.

## Features
- Solves valid 9x9 Sudoku boards
- Uses recursion and backtracking
- Includes row, column, and 3x3 block validation

## Concepts
- 2D lists
- Recursion
- Backtracking
- Constraint checking

## How to run
Run the Python file and enter the Sudoku board values when promted.

- Enter numbers from 0 to 9
- Use 0 for empty cells
- The program will output the solved board
